# Audio Promotional File

Place your promotional audio file here with the name `promotional.mp3`

This audio file will be used in the "Promotional Musical for Artistic and Cultural Events" section of the website.

## File Requirements:
- Format: MP3
- Filename: `promotional.mp3`
- Recommended quality: 192 kbps or higher
- Maximum file size: 10 MB (recommended)

Once you add the file, it will automatically be available in the promotional section of the website.
